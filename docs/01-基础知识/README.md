## 常见问题

### Q1: 光猫WiFi能满足什么场景？
**A**: 单人轻度使用、小户型简单需求、临时过渡。如果是多人合租、大户型、设备多，建议买路由器。

### Q2: 光猫有哪些功能？
**A**: 光电转换、PON鉴权、PPPoE拨号、路由和NAT、无线AP、有线交换机，一个设备集成了六大功能。

### Q3: 运营商网络和家庭网络有什么区别？
**A**: 运营商骨干网用SDH、OTN等专用技术，家庭网络用以太网。通过OLT、分光器、光猫逐步转换。

### Q4: 合租房怎么优化网络？
**A**: 参考第2集，买个路由器放在房屋中心位置，让光猫专心做光电转换和拨号。

### Q5: 为什么要向光猫宣战？
**A**: 光猫集成了太多功能（ONU + 交换机 + 路由器 + AP），性能羸弱。通过功能拆解，让专业设备做专业的事，可以大幅提升网络性能。
