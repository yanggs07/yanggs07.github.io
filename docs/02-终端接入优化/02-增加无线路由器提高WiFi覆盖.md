# 第2集：增加无线路由器提高WiFi覆盖

## 口播脚本

### 开场

大家好，上一集我们解决了小糖的有线网络问题，通过一个交换机，让PS5用上了稳定的有线连接。

但是新的问题又来了：小糖的室友小美搬进来了，住在另一个卧室。小美发现，她房间里的WiFi信号只有一格，刷视频经常卡顿。

光猫的WiFi覆盖不够了，怎么办？

今天我们就来聊聊，怎么通过增加一个无线路由器，来提高家里的WiFi覆盖。

---

### 第一部分：光猫WiFi的局限性

我们先来分析一下，为什么光猫的WiFi覆盖不够？

#### 1. 位置不好

光猫通常放在弱电箱里，而弱电箱一般在门口或者角落，不在房屋的中心位置。

WiFi信号是向四周发射的，如果光猫在角落，那信号就浪费了一半。

#### 2. 天线少

光猫通常只有2根天线，有些甚至是内置天线。

天线少，信号强度就弱，覆盖范围就小。

#### 3. 性能一般

光猫的主要任务是光电转换和拨号，WiFi只是附带功能。

所以光猫的WiFi芯片性能一般，处理能力有限，多个设备同时连接就会卡。

#### 4. 标准老旧

很多运营商配的光猫还是WiFi 5（802.11ac），甚至WiFi 4（802.11n）。

现在都2026年了，WiFi 6（802.11ax）已经普及，WiFi 7（802.11be）都开始出现了。

**所以，光猫WiFi适合单人小户型，但如果是多人合租、大户型，就需要专业的无线路由器了。**

---

### 第二部分：路由器的选择

小糖决定买一个无线路由器。但是打开京东一看，路由器从100元到3000元都有，怎么选呢？

#### 看WiFi标准

**WiFi 4（802.11n）**：
- 最高速度300Mbps
- 已经淘汰，不建议购买

**WiFi 5（802.11ac）**：
- 最高速度1300Mbps
- 目前还在用，但不推荐新买

**WiFi 6（802.11ax）**：
- 最高速度9.6Gbps（理论值）
- 实际速度1200-2400Mbps
- 支持更多设备同时连接
- **目前主流，强烈推荐**

**WiFi 6E**：
- 支持6GHz频段
- 干扰更少，速度更快
- 设备支持度还不高，价格贵

**WiFi 7（802.11be）**：
- 最高速度46Gbps（理论值）
- 2024年才开始商用
- 价格很贵，设备支持度低
- 可以观望，不急着买

**小糖的选择**：WiFi 6路由器，性价比最高。

#### 看频段

路由器有单频、双频、三频之分：

**单频（2.4GHz）**：
- 穿墙能力强，覆盖范围大
- 速度慢，干扰多
- 已经淘汰

**双频（2.4GHz + 5GHz）**：
- 2.4GHz负责覆盖，5GHz负责速度
- 目前主流
- **推荐选择**

**三频（2.4GHz + 5GHz + 5GHz/6GHz）**：
- 多一个频段，可以减少干扰
- 适合设备特别多的场景
- 价格贵，一般家庭用不上

**小糖的选择**：双频路由器就够了。

#### 看天线数量

天线越多，信号越强，覆盖越好。

- **2天线**：小户型够用
- **4天线**：中户型推荐
- **6天线或更多**：大户型或追求极致覆盖

**小糖的选择**：4天线，覆盖三室一厅够用。

#### 看处理器和内存

路由器也有CPU和内存，性能越好，处理能力越强。

- **入门级**：单核CPU，128MB内存
- **中端**：双核CPU，256MB内存
- **高端**：四核CPU，512MB或1GB内存

多设备同时连接、下载、游戏，都需要路由器的处理能力。

**小糖的选择**：双核CPU，256MB内存，够用。

#### 看端口

- **WAN口**：连接光猫或上级网络，通常1个
- **LAN口**：连接有线设备，通常4个
- **USB口**：可以接U盘、打印机，可选

**重要**：WAN口和LAN口都要是**千兆口**，不要买百兆的。

#### 小糖的最终选择

小糖综合考虑，选择了**小米AX3000**：

- WiFi 6标准
- 双频（2.4GHz + 5GHz）
- 4天线
- 双核CPU，256MB内存
- 千兆WAN口 + 3个千兆LAN口
- 价格：300元左右

性价比很高，适合大多数家庭。

### 第三部分：光猫和路由器的配合方式

买回路由器，小糖面临一个问题：光猫和路由器怎么连接？

有两种方式：

#### 方式一：光猫路由模式 + 路由器路由模式（双路由）

**连接方式**：
- 光猫继续拨号，继续做路由
- 路由器的WAN口连接光猫的LAN口
- 路由器也做路由，形成二级路由

**优点**：
- 简单，插上就能用
- 不需要配置光猫

**缺点**：
- 双重NAT，可能影响游戏、远程访问
- 光猫和路由器都在做路由，浪费性能
- 网络结构复杂，排查问题麻烦

**适合场景**：
- 临时应急
- 不想折腾光猫配置
- 对网络要求不高

#### 方式二：光猫桥接模式 + 路由器路由模式（单路由）

**连接方式**：
- 光猫改成桥接模式，只做光电转换
- 路由器的WAN口连接光猫的LAN口
- 路由器拨号，路由器做路由

**优点**：
- 只有一层NAT，性能更好
- 网络结构简单，排查问题容易
- 路由器性能更好，功能更强

**缺点**：
- 需要配置光猫，稍微麻烦一点
- 需要知道宽带账号密码

**适合场景**：
- 追求性能和稳定性
- 需要公网IP
- 需要端口转发、远程访问

**小糖的选择**：方式二，光猫桥接 + 路由器拨号。

虽然稍微麻烦一点，但是性能更好，网络结构更清晰。

### 第四部分：光猫桥接配置

小糖开始配置光猫桥接模式。

#### 第1步：登录光猫管理界面

- 电脑或手机连接光猫的WiFi
- 浏览器输入：192.168.1.1
- 输入光猫的管理账号密码

**问题**：光猫的管理账号密码是什么？

- 有些光猫背面有标签，写着账号密码
- 有些需要打运营商客服电话，要超级管理员账号
- 电信：账号通常是 useradmin，密码需要客服提供
- 联通：账号通常是 CUAdmin，密码需要客服提供
- 移动：账号通常是 CMCCAdmin，密码需要客服提供

小糖打了10000号，客服给了超级管理员账号密码。

#### 第2步：找到上网配置

登录后，找到"网络" → "宽带设置" 或 "WAN配置"。

会看到一个连接列表，通常有一个连接是"已连接"状态，这就是当前的拨号连接。

#### 第3步：修改连接模式

- 找到当前的拨号连接，点击"编辑"
- 把"连接模式"从"路由"改成"桥接"
- 保存

有些光猫需要删除原来的路由连接，新建一个桥接连接。具体操作看光猫型号，不同品牌略有差异。

#### 第4步：关闭光猫WiFi（可选）

既然要用路由器的WiFi，光猫的WiFi就可以关掉了，省电也减少干扰。

找到"无线设置" → "关闭WiFi"。

#### 第5步：保存并重启

保存配置，重启光猫。

重启后，光猫就变成桥接模式了，不再拨号，只做光电转换。

---

### 第五部分：路由器拨号配置

光猫配置好了，现在配置路由器。

#### 第1步：连接路由器

- 用网线把路由器的WAN口连接到光猫的LAN口
- 路由器通电
- 电脑或手机连接路由器的WiFi（初始WiFi名称和密码在路由器背面）

#### 第2步：登录路由器管理界面

- 浏览器输入：192.168.31.1（小米路由器）或 192.168.1.1（其他品牌）
- 首次登录会进入设置向导

#### 第3步：选择上网方式

路由器会自动检测上网方式，选择"PPPoE拨号"。

#### 第4步：输入宽带账号密码

输入运营商给的宽带账号和密码。

这个账号密码就是装宽带时运营商给的，如果忘了，可以打客服电话查询。

#### 第5步：设置WiFi名称和密码

- WiFi名称：XiaoTang_Home（自己喜欢的名字）
- WiFi密码：设置一个强密码，至少8位，包含字母数字符号
- 2.4GHz和5GHz可以设置成同一个名称，路由器会自动切换（推荐）

#### 第6步：设置管理密码

设置路由器的管理密码，以后登录管理界面需要用。

可以和WiFi密码一样，也可以不一样。

#### 第7步：完成配置

保存配置，路由器开始拨号。

等待30秒左右，路由器拨号成功，就可以上网了。

### 第六部分：测试和优化

配置完成，小糖开始测试。

#### 测试覆盖

小糖拿着手机，在家里各个房间走了一圈：

- **客厅**：信号满格，速度300Mbps
- **主卧（小糖房间）**：信号满格，速度280Mbps
- **次卧（小美房间）**：信号3格，速度150Mbps
- **厨房**：信号2格，速度80Mbps
- **卫生间**：信号1格，速度30Mbps

比光猫WiFi好多了，但是次卧和厨房还是有点弱。

#### 优化位置

小糖把路由器从弱电箱移到了客厅电视柜上，位置更居中。

再测试一遍：

- **客厅**：信号满格，速度300Mbps
- **主卧**：信号满格，速度280Mbps
- **次卧**：信号满格，速度250Mbps ✅
- **厨房**：信号3格，速度120Mbps ✅
- **卫生间**：信号2格，速度60Mbps ✅

好多了！小美也很满意，房间里终于可以流畅刷视频了。

#### 优化信道

小糖发现，晚上高峰期，WiFi速度会变慢。

她登录路由器管理界面，发现2.4GHz信道是"自动"，5GHz信道也是"自动"。

小糖下载了一个WiFi分析工具（WiFi Analyzer），扫描了一下周围的WiFi：

- 2.4GHz：1、6、11信道都很拥挤
- 5GHz：36、40、44信道比较空

小糖手动设置：
- 2.4GHz：信道11（相对不那么拥挤）
- 5GHz：信道149（很空，干扰少）

设置完后，晚上高峰期速度明显提升。

### 第七部分：路由器的进阶功能

路由器配置好了，我们再聊聊路由器的一些进阶功能。

#### QoS（服务质量）

QoS可以给不同的设备或应用分配优先级。

比如：
- 游戏主机优先级最高
- 视频会议优先级高
- 下载优先级低

这样即使有人在下载，也不会影响游戏和视频会议。

小米路由器的QoS叫"游戏加速"，可以自动识别游戏流量，给予优先级。

#### 访客网络

访客网络可以给客人提供WiFi，但是隔离内网设备。

这样客人可以上网，但是访问不了你的NAS、打印机等内网设备，更安全。

#### 家长控制

可以限制孩子的上网时间，或者屏蔽某些网站。

比如：
- 工作日晚上10点后断网
- 周末可以上网，但不能访问游戏网站

#### 端口转发

如果你有NAS、服务器，需要从外网访问，就需要配置端口转发。

比如：
- 外网访问 你的公网IP:8080
- 路由器转发到 内网NAS:8080

这样就可以在外面访问家里的NAS了。

#### DDNS（动态域名）

如果你没有固定公网IP，每次重启光猫IP都会变，可以用DDNS服务。

DDNS会把你的域名（比如 xiaotang.ddns.net）自动指向你当前的公网IP。

这样你就可以用域名访问家里的设备，不用记IP地址。

### 结尾总结

好，今天我们讲了怎么通过增加无线路由器来提高WiFi覆盖。简单总结一下：

1. **光猫WiFi的局限性**：位置不好、天线少、性能一般、标准老旧
2. **路由器的选择**：WiFi 6、双频、4天线、千兆口，300-500元性价比最高
3. **光猫和路由器的配合**：推荐光猫桥接 + 路由器拨号，性能更好
4. **配置步骤**：光猫改桥接 → 路由器拨号 → 设置WiFi → 测试优化
5. **进阶功能**：QoS、访客网络、家长控制、端口转发、DDNS

小糖通过一个300元的路由器，解决了WiFi覆盖问题，全家都能流畅上网了。

但是，如果户型更大，或者有多层楼，单个路由器可能还是覆盖不够。下一集，我们来聊聊怎么通过部署独立AP，来实现更好的无线覆盖。
