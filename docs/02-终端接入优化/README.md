## 方案对比

| 方案 | 优点 | 缺点 | 适合场景 |
|------|------|------|----------|
| 光猫WiFi | 零成本、即插即用 | 性能一般、覆盖有限 | 单人小户型 |
| 单路由器 | 性能好、成本低 | 覆盖有限 | 中小户型 |
| 路由器+AP | 覆盖好、有线稳定 | 需要布线 | 大户型、已布线 |
| Mesh组网 | 覆盖好、无需布线 | 无线回程性能损失 | 大户型、不便布线 |
| AC+AP | 统一管理、稳定性高 | 成本高、配置复杂 | 别墅、商用场景 |

## 常见问题

### Q1: 光猫WiFi不够用，是买路由器还是买AP？
**A**: 如果只是单点覆盖不足，买路由器更划算。如果是大户型需要多点覆盖，且已经布线，可以考虑AP。

### Q2: Mesh和AC+AP有什么区别？
**A**: Mesh主要用无线回程，适合不方便布线的场景。AC+AP用有线回程，性能更稳定，但需要预先布线。

### Q3: 有线回程和无线回程哪个好？
**A**: 有线回程性能更好、更稳定，但需要布线。无线回程灵活方便，但会损失一半带宽。

### Q4: 需要买多贵的路由器？
**A**: 看户型和需求。小户型300-500元足够，大户型建议800-1500元，追求极致体验可以考虑2000元以上。
