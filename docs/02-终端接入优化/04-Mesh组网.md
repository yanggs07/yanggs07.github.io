# 第4集：Mesh组网 - 无线路由器的扩展

## 口播脚本

### 开场

大家好，上一集我们讲了怎么通过部署独立AP来提高WiFi覆盖，但前提是家里有预埋网线，可以用有线回程。

但是很多人家里装修时没有预埋网线，或者已经装修好了，不想破坏墙面重新布线。这时候怎么办呢？

今天我们就来聊聊Mesh组网，通过无线回程实现全屋WiFi覆盖。

### 第一部分：什么是Mesh

#### Mesh的定义

Mesh的中文叫"网状网络"或"分布式路由"。

简单说，Mesh就是**多个路由器组成一个网络，互相无线连接，共同提供WiFi覆盖**。

传统的网络是"星型"结构：
```
        路由器
       /  |  \
      /   |   \
   设备1 设备2 设备3
```

Mesh网络是"网状"结构：
```
   路由器1 ←→ 路由器2
      ↕         ↕
   路由器3 ←→ 路由器4
      ↕         ↕
    设备      设备
```

每个路由器（Mesh节点）都可以和其他节点通信，形成一个网状网络。

#### Mesh的特点

**1. 无需布线**
- 节点之间通过WiFi连接
- 插电就能用，部署简单

**2. 自动组网**
- 节点会自动发现彼此
- 自动选择最佳路径
- 某个节点故障，自动切换到其他路径

**3. 统一管理**
- 所有节点用同一个APP管理
- 统一的WiFi名称和密码
- 无缝漫游

**4. 扩展灵活**
- 覆盖不够，加一个节点就行
- 不需要重新配置

#### Mesh vs 传统WiFi扩展器

很多人会把Mesh和WiFi扩展器（中继器）混淆，我们来说清楚：

**传统WiFi扩展器**：
- 接收路由器的WiFi信号，再发射出去
- 速度减半（一半接收，一半发射）
- 需要不同的WiFi名称，或者手动切换
- 稳定性差，容易掉线

**Mesh系统**：
- 专门设计的多节点系统
- 有专用的回程频段，速度损失小
- 统一的WiFi名称，自动漫游
- 稳定性好，智能路由

**简单说**：Mesh是专业的多节点WiFi系统，WiFi扩展器是廉价的信号放大器。

### 第三部分：Mesh的工作原理

在选购Mesh之前，我们先了解一下Mesh是怎么工作的。

#### 三频Mesh vs 双频Mesh

Mesh路由器有双频和三频之分：

**双频Mesh（2.4GHz + 5GHz）**：
- 2.4GHz和5GHz都用于设备连接和节点回程
- 回程会占用一部分带宽
- 速度损失较大（约50%）
- 价格便宜

**三频Mesh（2.4GHz + 5GHz + 5GHz）**：
- 2.4GHz和一个5GHz用于设备连接
- 另一个5GHz专门用于节点回程
- 速度损失小（约20-30%）
- 价格贵

**举个例子**：

双频Mesh：
- 路由器1的5GHz：50%给设备，50%给回程
- 路由器2的5GHz：50%接收回程，50%给设备
- 实际速度：原来的25-50%

三频Mesh：
- 路由器1的5GHz-1：100%给设备
- 路由器1的5GHz-2：100%给回程
- 路由器2的5GHz-1：100%给设备
- 实际速度：原来的70-80%

**所以，如果预算够，强烈推荐三频Mesh。**

#### 有线回程 vs 无线回程

即使是Mesh系统，也可以用有线回程：

**有线回程**：
- 节点之间用网线连接
- 速度快，稳定
- 无线频段100%用于设备连接
- 需要布线

**无线回程**：
- 节点之间用WiFi连接
- 速度有损失，但比传统扩展器好
- 不需要布线
- 部署灵活

**最佳实践**：能用有线回程就用有线回程，实在不行再用无线回程。

#### 星型 vs 菊花链

Mesh节点的连接方式有两种：

**星型（Star）**：
```
      主路由
     /     \
   节点1   节点2
```
- 所有节点都直接连接主路由
- 速度快，延迟低
- 但是节点必须在主路由的覆盖范围内

**菊花链（Daisy Chain）**：
```
主路由 → 节点1 → 节点2
```
- 节点可以串联
- 可以覆盖更远的距离
- 但是每多一跳，速度和延迟都会变差

**最佳实践**：尽量用星型，避免菊花链超过2跳。

小明开始选购Mesh路由器。

#### 看频段

**双频Mesh**：
- 价格：300-600元（2只装）

**三频Mesh**：

- 适合：大户型、追求性能
- 代表：华硕ZenWiFi AX、网件Orbi

**小明的选择**：三频Mesh，因为户型大，追求性能。

#### 看WiFi标准

和路由器一样，Mesh也有WiFi标准：

- **WiFi 5 Mesh**：不推荐新买
- **WiFi 6 Mesh**：主流，推荐
- **WiFi 6E Mesh**：高端，价格贵

**小明的选择**：WiFi 6 Mesh。

#### 看节点数量

Mesh通常是套装销售：

- **2只装**：适合100平米以内
- **3只装**：适合100-150平米
- **单只装**：可以后续扩展

**小明的选择**：2只装，如果不够再买一只扩展。

#### 看品牌

**入门级（300-600元）**：
- 小米、TP-Link、华为
- 适合预算有限的用户

**中端（600-1200元）**：
- 华硕、网件、领势
- 性能更好，功能更多

**高端（1200元以上）**：
- 网件Orbi、华硕ZenWiFi、Eero
- 企业级性能，适合别墅

**小明的选择**：华硕ZenWiFi AX，1200元（2只装），三频WiFi 6。

### 第五部分：Mesh的部署

小明买回Mesh路由器，开始部署。

#### 第1步：确定主节点位置

主节点（主路由）通常放在光猫旁边，因为要连接光猫。

小明把主节点放在客厅电视柜旁边，连接光猫。

#### 第2步：配置主节点

**方式一：通过手机APP配置（推荐）**

1. 主节点连接光猫，通电
2. 手机下载华硕路由器APP
3. 打开APP，扫描主节点
4. 选择上网方式：PPPoE拨号
5. 输入宽带账号密码
6. 设置WiFi名称和密码
7. 完成配置

**方式二：通过网页配置**

1. 电脑或手机连接主节点的WiFi
2. 浏览器输入：router.asus.com
3. 按照向导配置

小明用APP配置，5分钟搞定。

#### 第3步：确定子节点位置

子节点的位置很重要，直接影响覆盖效果。

**原则**：
- 子节点要在主节点的覆盖范围内（信号至少2格）
- 子节点要能覆盖到需要覆盖的区域
- 避免子节点之间距离太远

小明用手机测试了一下：
- 主卧：信号1格，需要子节点
- 走廊：信号3格，可以放子节点
- 次卧：信号1格，需要子节点

小明决定：
- 子节点1：放在走廊，覆盖主卧和走廊
- 如果次卧还是信号弱，再买一个子节点

#### 第4步：添加子节点

1. 子节点放在走廊，通电
2. 打开APP，点击"添加节点"
3. APP自动搜索子节点
4. 找到后，点击"添加"
5. 等待1-2分钟，子节点自动配置完成

非常简单，不需要手动配置WiFi名称和密码，子节点会自动同步主节点的配置。

#### 第5步：测试覆盖

配置完成，小明拿着手机测试：

- **客厅**：信号满格，速度300Mbps
- **主卧**：信号满格，速度220Mbps ✅
- **次卧**：信号3格，速度150Mbps ✅
- **厨房**：信号满格，速度250Mbps
- **卫生间**：信号3格，速度120Mbps

比单个路由器好多了！主卧和次卧都能流畅上网了。

#### 第6步：测试漫游

小明拿着手机，从客厅走到主卧：

- 在客厅，手机连接主节点
- 走到走廊，信号开始切换
- 走到主卧，手机自动切换到子节点
- 全程没有断线，视频播放流畅

完美！

### 第六部分：Mesh的优化

Mesh部署好了，我们再聊聊一些优化技巧。

#### 优化节点位置

如果覆盖效果不理想，可以调整节点位置。

**工具**：用手机APP查看节点之间的连接质量。

- **绿色**：连接质量好
- **黄色**：连接质量一般，可以优化
- **红色**：连接质量差，需要调整位置

如果子节点显示红色或黄色，说明离主节点太远，需要移近一点。

#### 优化回程频段

有些Mesh系统可以手动选择回程频段：

- **自动**：系统自动选择最佳频段
- **5GHz-1**：手动指定用5GHz-1做回程
- **5GHz-2**：手动指定用5GHz-2做回程

一般选"自动"就好，系统会智能选择。

#### 优化信道

和路由器一样，Mesh也可以优化信道。

用WiFi分析工具扫描周围的WiFi，选择干扰少的信道。

不过大部分Mesh系统会自动优化信道，不需要手动设置。

#### 固件升级

定期升级Mesh系统的固件，可以获得更好的性能和稳定性。

大部分Mesh系统会自动检查更新，也可以手动在APP里升级。

### 第七部分：Mesh的进阶功能

#### 有线回程模式

如果后来布了网线，可以把Mesh改成有线回程模式。

只需要用网线连接主节点和子节点，Mesh系统会自动切换到有线回程，性能大幅提升。

#### 访客网络

Mesh系统通常支持访客网络，可以给客人提供WiFi，但隔离内网设备。

#### 家长控制

可以限制孩子的上网时间，或者屏蔽某些网站。

#### IoT网络

有些Mesh系统支持单独的IoT网络（2.4GHz），专门给智能家居设备用。

这样可以把IoT设备和主网络隔离，更安全。

#### 网络优先级

可以给某些设备设置优先级，比如游戏主机、工作电脑优先级高，下载设备优先级低。

---

### 第八部分：Mesh的局限性

Mesh虽然方便，但也有一些局限性：

#### 1. 速度损失

无线回程会有速度损失：
- 双频Mesh：损失50%左右
- 三频Mesh：损失20-30%

如果追求极致性能，还是有线回程更好。

#### 2. 延迟增加

每多一跳，延迟就会增加5-10ms。

如果是菊花链3跳，延迟可能增加20-30ms，对游戏体验有影响。

#### 3. 稳定性不如有线

无线回程容易受干扰，稳定性不如有线回程。

如果周围WiFi很多，或者有微波炉等干扰源，可能会影响Mesh的性能。

#### 4. 价格较贵

Mesh系统比单个路由器贵，尤其是三频Mesh。

如果预算有限，可以考虑双频Mesh，或者等有线回程的AP方案。

#### 5. 不能混搭

不同品牌的Mesh系统通常不能混搭，必须买同一品牌的节点。

有些品牌甚至不同型号也不能混搭，购买前要确认好。

### 第九部分：Mesh vs AP，怎么选？

很多人会纠结：到底是买Mesh还是买AP？

我们来对比一下：

| 对比项 | Mesh | AP |
|--------|------|-----|
| 布线 | 不需要 | 需要 |
| 速度 | 有损失（20-50%） | 无损失 |
| 稳定性 | 一般 | 好 |
| 延迟 | 稍高 | 低 |
| 部署难度 | 简单 | 稍复杂 |
| 扩展性 | 好 | 好 |
| 价格 | 较贵 | 便宜 |

**选择建议**：

**选Mesh的场景**：
- 已经装修好，无法布线
- 租房，不想破坏墙面
- 追求部署简单
- 对性能要求不是特别高

**选AP的场景**：
- 装修时预留了网线
- 可以明线布线
- 追求极致性能和稳定性
- 预算有限

**最佳方案**：
- 装修时预埋网线到各个房间
- 买Mesh系统，用有线回程
- 这样既有Mesh的便捷管理，又有有线的性能和稳定性

### 结尾总结

好，今天我们讲了Mesh组网。简单总结一下：

1. **什么是Mesh**：多个路由器组成网状网络，无线互联，统一管理
2. **Mesh的特点**：无需布线、自动组网、统一管理、扩展灵活
3. **三频 vs 双频**：三频有专用回程频段，速度损失小，推荐
4. **Mesh的部署**：主节点连光猫、子节点放中间位置、APP自动配置
5. **Mesh的局限性**：速度有损失、延迟稍高、价格较贵
6. **Mesh vs AP**：Mesh适合无法布线的场景，AP适合有线回程的场景

小明通过一套1200元的三频Mesh系统，解决了全屋WiFi覆盖问题，不需要破坏墙面布线。
