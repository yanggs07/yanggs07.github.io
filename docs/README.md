# 家庭网络系列短视频脚本库

## 📚 项目简介

这是一个家庭网络系列短视频脚本库，通过真实场景和实战案例，帮助普通用户从零开始，理解和优化家庭网络。

## 🎯 目标受众

- 第一次装宽带的用户
- 合租房、出租屋用户
- 网络小白，想了解家庭网络基础知识
- 遇到网络问题，需要排查和优化
- 智能家居爱好者，需要规划网络架构

## 📖 系列结构

### [第一季：基础知识篇](./01-基础知识/README.md) (4集)
从全局视角理解家庭网络，通过真实场景建立完整认知

- ✅ 第1集：网络体验的秘密 - 从终端到云端的完整链路
- ✅ 第2集：第一次装宽带 - 从零开始的家庭网络
- ✅ 第3集：光纤入户与光猫 - FTTH、FTTR和ONU详解
- 📝 第4集：折腾家庭网络就是要向光猫宣战 - 功能拆解与优化策略

### [第二季：终端接入优化](./02-终端接入优化/README.md) (5集)
从有线到无线，全面提升家庭网络覆盖和性能

- ✅ 第1集：有线网络接入与交换机 - 多设备有线接入方案
- ✅ 第2集：增加无线路由器提高WiFi覆盖 - 单路由器优化
- ✅ 第3集：独立AP提高无线覆盖 - 有线回程多AP方案
- ✅ 第4集：Mesh组网 - 无线回程全屋覆盖
- ✅ 第5集：AC+AP模式提高无线覆盖 - 企业级统一管理

### 后续内容规划中...
- 第三季：网络性能优化
- 第四季：网络安全与管理
- 第五季：智能家居网络规划

## 🎬 内容特色

### 真实场景
- 以"小糖第一次装宽带"为主线
- 展示真实的问题和解决过程
- 让观众产生强烈的代入感

### 全局视角
- 不只讲设备，更讲整个网络链路
- 理解木桶效应，找到真正的瓶颈
- 明确哪些能控制，哪些能优化

### 实战为主
- 每集都有实际操作演示
- 提供详细的配置步骤
- 分享常见问题的解决方案

### 循序渐进
- 从全局认知到具体实战
- 前后内容相互关联
- 建立完整的知识体系

## 📊 制作规范

### 视频规格
- **时长**: 8-10分钟/集
- **分辨率**: 1080P (1920x1080)
- **帧率**: 30fps
- **格式**: MP4 (H.264)

### 脚本结构
1. **开场** (30秒): 痛点引入 + 本集主题
2. **知识讲解** (2-3分钟): 核心概念 + 原理说明
3. **实战演示** (4-6分钟): 操作步骤 + 注意事项
4. **总结回顾** (30秒): 要点总结 + 下集预告

### 视觉呈现
- 使用动画演示抽象概念（网络链路、信号覆盖等）
- 实拍设备和操作界面
- 配合图表和示意图
- 关键步骤添加字幕标注

## 🛠️ 使用指南

### 脚本编写流程
1. 确定本集主题和目标
2. 列出核心知识点
3. 设计演示案例
4. 编写详细脚本
5. 准备拍摄清单
6. 审核和优化

### 文件命名规范
- 文件夹: `序号-名称` (如: `01-基础知识`)
- 脚本文件: `序号-标题.md` (如: `01-网络体验的秘密.md`)
- 素材文件: `集数-素材类型-描述` (如: `01-动画-网络链路`)

### 版本管理
- 使用Git进行版本控制
- 每完成一集脚本提交一次
- 重大修改使用分支开发
